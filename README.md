# cheatsheets
This repository contains cheat sheets for use with [Dash](https://kapeli.com/dash).

## Installation

- Prerequisites: [Ruby](https://www.ruby-lang.org), [Bundler](http://bundler.io)
- Run `bundle install`

## Usage

```
$ git clone git@github.com:Shark/cheatsheets.git
$ ./build_all
```
